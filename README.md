# Portfolio Project

This is a portfolio project repository using Git Flow workflow.

## Git Flow Workflow

This repository follows the Git Flow workflow with the following branches:

- `main` - Production branch
- `develop` - Development branch
- `feature/*` - Feature branches
- `release/*` - Release branches
- `hotfix/*` - Hotfix branches

## Getting Started

1. Clone the repository
2. Install dependencies (if any)
3. Start development

## Development

1. Create a new feature branch from `develop`
2. Make your changes
3. Create a pull request to merge back into `develop`

## Contributing

Please follow the Git Flow workflow when contributing to this project.
